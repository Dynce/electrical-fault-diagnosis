About the System

The Electrical Fault Diagnosis System is an intelligent, web-based application designed to assist technicians, engineers, and maintenance professionals in diagnosing electrical faults efficiently and accurately.

Using advanced machine learning algorithms and real-time sensor data analysis, the system can identify various electrical faults including overheat conditions, overload situations, and short circuits. This enables quick decision-making and preventive maintenance.

Technology Stack

The system is built using modern web technologies and machine learning frameworks:

  • Backend: Python, Flask, Flask-Login, Flask-Mail
  • Machine Learning: Scikit-learn, RandomForest Classifier, StandardScaler
  • Frontend: HTML5, CSS3, JavaScript
  • Database: SQLite3 with role-based access control
  • Security: Password hashing, Email verification, Token-based authentication
  • Deployment: Waitress WSGI server
